------------------------------------------------------------------------
This project models a partially written program to support a taxi
company that operates both taxis and shuttles.
Taxis can be booked to any single destination.
Shuttles operate around a circular route consisting of a limited list
of destinations.
------------------------------------------------------------------------

PROJECT TITLE: taxi-company
PURPOSE OF PROJECT: to illustrate possibilities for inheritance
VERSION or DATE: 2010.12.03
HOW TO START THIS PROJECT:
    Create a TaxiCo object. Then call its addTaxi and addShuttle methods
    to add an arbitrary number of these vehicles.
    The showStatus method lists the vehicles in the fleet.
    The getTaxi method allows a Taxi to be looked up and placed on the
    object bench if required.
